## 4. 数据清洗与质量控制

### 4.1 清洗流程设计

数据清洗遵循「先诊断、后处理、再验证」的三阶段流程：

```
原始数据 → 质量诊断 → 清洗处理 → 质量验证 → 干净数据
```

### 4.2 缺失值处理策略

| 缺失类型 | 处理策略 | 原因说明 |
|----------|----------|----------|
| 节假日缺失 | 前向填充（ffill） | 不同市场交易日历差异 |
| 单点缺失 | 线性插值 | 数据传输偶发中断 |
| 连续缺失>5天 | 记录并标记 | 可能为停牌或数据异常 |

**本次数据缺失情况：**
- 原始缺失值：由节假日导致，七只股票同步缺失
- 处理方式：前向填充（forward fill）
- 填充后缺失率：0.00%

### 4.3 收益率序列计算

从复权价格计算日度收益率序列：

$$R_t = \frac{P_t - P_{t-1}}{P_{t-1}}$$

其中：
- $R_t$：第t日收益率
- $P_t$：第t日复权收盘价
- $P_{t-1}$：第t-1日复权收盘价

**首行处理：** 因计算收益率需前一日价格，首行产生NA值，予以剔除。

### 4.4 数据落库管理

清洗后的数据存入SQLite数据库，便于后续查询与扩展：

| 表名 | 内容说明 |
|------|----------|
| prices_clean | 清洗后的日度收盘价 |
| returns | 日度收益率序列 |
| metrics | 计算好的风险收益指标 |
| correlation_matrix | 收益率相关系数矩阵 |

## 5. 指标构建与计算口径

### 5.1 收益类指标

#### 年化收益率（Annualized Return）

$$\text{年化收益率} = \bar{R}_{\text{daily}} \times 252$$

其中：
- $\bar{R}_{\text{daily}}$：日度收益率样本均值
- 252：年化交易日数（美国市场惯例）

#### 年度收益率（Yearly Return）

$$R_{\text{year}} = \prod_{t=1}^{T}(1 + R_t) - 1$$

通过年度重采样计算历年收益表现。

### 5.2 风险类指标

#### 年化波动率（Annualized Volatility）

$$\sigma_{\text{annual}} = \sigma_{\text{daily}} \times \sqrt{252}$$

其中：
- $\sigma_{\text{daily}}$：日度收益率样本标准差
- 使用修正标准差（ddof=1）

#### 最大回撤（Maximum Drawdown）

最大回撤衡量投资从峰值到谷底的最大跌幅：

$$\text{DD}_t = \frac{V_t - \max(V_{1:t})}{\max(V_{1:t})}$$

$$\text{MDD} = \min(\text{DD}_t)$$

其中：
- $V_t$：累计收益序列（$(1+R)$的累积乘积）
- $\max(V_{1:t})$：截至t日的历史最高值

### 5.3 风险调整后收益指标

#### 夏普比率（Sharpe Ratio）

$$\text{Sharpe} = \frac{R_p - R_f}{\sigma_p}$$

参数设置：
- $R_p$：年化收益率
- $R_f$：无风险利率 = 4.5%（当前美国短期国债水平）
- $\sigma_p$：年化波动率

#### 超额收益（Alpha vs 标普500）

$$\alpha = R_{\text{stock}} - R_{\text{SP500}}$$

计算各股票相对标普500指数的超额收益水平。

### 5.4 估值指标

| 指标 | 计算公式 | 含义说明 |
|------|----------|----------|
| PE（市盈率） | 股价 / 每股收益 | 市场对公司盈利能力的定价倍数 |
| PB（市净率） | 股价 / 每股净资产 | 市场对公司净资产的溢价程度 |
| 市值 | 股价 × 流通股数 | 公司市场价值规模 |
| 营收增速 | (本期营收 - 上期营收) / 上期营收 | 成长能力 |
| 毛利率 | 毛利 / 营收 | 核心业务盈利能力 |
| ROE | 净利润 / 净资产 | 股东权益回报率 |

## 9. 局限性说明与后续研究方向

### 9.1 本研究局限性

| 局限性类别 | 具体说明 |
|------------|----------|
| **数据来源局限** | yfinance财务字段口径因公司而异，不同公司会计处理存在差异 |
| **历史数据时滞** | 财务数据存在更新时滞，可能无法反映最新季度业绩 |
| **样本期局限** | 5年样本期较短，未覆盖完整经济周期（如2008金融危机） |
| **指标单一性** | 仅使用价格数据，未纳入期权市场隐含波动率等信息 |
| **汇率因素** | 未考虑美元兑人民币汇率对人民币投资者的实际收益影响 |
| **分红处理** | 使用复权价格，已隐含分红再投资，但未单独分析股息收益率 |

### 9.2 后续研究方向

1. **多因子模型扩展**
   - 加入Fama-French三因子/五因子分析
   - 纳入动量因子、质量因子等风格因子暴露

2. **事件驱动分析**
   - 财报发布前后超额收益分析
   - 美联储议息会议对板块影响
   - AI相关政策/技术突破事件影响

3. **高频数据研究**
   - 日内波动率模式分析
   - 期权市场隐含波动率与实现波动率对比

4. **国际比较视角**
   - 与A股科技板块（如科创50）对比
   - 与欧洲科技板块（如STOXX Europe 600 Technology）对比

5. **ESG整合分析**
   - 纳入ESG评级数据
   - 分析ESG因子与风险收益的关系
